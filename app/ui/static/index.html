With App Engine you don't upload code without having to worry about configuring individual servers to handle it. However when requests come hit App Engine, it needs to scale up workers to run your code and handle the request. Each one of the figures here represents a worker, and the number below them is the number of requests each one handled.

App Engine can scale up to handle large amounts of load. With this game, we'll scale up depending on how hard you can swing this hammer. The harder you hit, the more requests we'll send at App Engine, which will lead to more workers being spun up.
